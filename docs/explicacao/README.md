# Explicacao Do Projeto

Esta pasta reune explicacoes rapidas para entender o projeto sem precisar procurar em muitos arquivos.

## Arquivos Desta Pasta

- Projeto geral (`PROJETO_GERAL.md`)
  Visao geral da estrutura e do fluxo do sistema.
- Backend (`BACKEND.md`)
  Explica como o backend esta organizado hoje.
- Frontend (`FRONTEND.md`)
  Explica a parte publica do frontend.
- Admin (`ADMIN.md`)
  Explica o painel administrativo.
- Rotas e componentes (`ROTAS_E_COMPONENTES.md`)
  Mapa de rota para componente.

## Ordem Recomendada

1. Ler Projeto geral (`PROJETO_GERAL.md`)
2. Ler Frontend (`FRONTEND.md`)
3. Ler Admin (`ADMIN.md`)
4. Ler Backend (`BACKEND.md`)
5. Consultar Rotas e componentes (`ROTAS_E_COMPONENTES.md`) quando quiser achar uma pagina
