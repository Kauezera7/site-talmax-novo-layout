# Explicacao Do Projeto

Esta pasta reúne explicações rápidas para entender o projeto sem precisar caçar informação em muitos arquivos.

## Arquivos Desta Pasta

- `PROJETO_GERAL.md`
  Visão geral da estrutura e do fluxo do sistema.
- `BACKEND.md`
  Explica como o backend está organizado hoje.
- `FRONTEND.md`
  Explica a parte pública do frontend.
- `ADMIN.md`
  Explica o painel administrativo.
- `ROTAS_E_COMPONENTES.md`
  Mapa de rota para componente.

## Ordem Recomendada

1. Ler `PROJETO_GERAL.md`
2. Ler `FRONTEND.md`
3. Ler `ADMIN.md`
4. Ler `BACKEND.md`
5. Consultar `ROTAS_E_COMPONENTES.md` quando quiser achar uma página
